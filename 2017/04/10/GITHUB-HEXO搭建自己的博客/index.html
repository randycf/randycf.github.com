无法访问本地网站

Q: 执行hexo server 提示访问 http://localhost:4000/, 但在浏览器无法打开, 一直等待,命令行也没有错误提示
A: 可能是4000端口被其他程序占用, 尝试修改服务器端口 hexo s -p 5000, 看能否正常访问http://localhost:5000/, 如果可以则确定是由于端口被占用的问题.
